Box Model

Core Rule
Every element is a rectangular box: content → padding → border → margin. Use box-sizing: border-box so padding/border don't add to declared width.

Classes (OOP in JS)
JavaScript
class Animal {
  #name;  // private field

  constructor(name) {
    this.#name = name;
  }

  speak() {
    return `${this.#name} makes a sound`;
  }

  get name() { return this.#name; }

  static create(name) {
    return new Animal(name);
  }
}

class Dog extends Animal {
  speak() { return "Woof!"; }
}

const d = new Dog("Rex");
d.speak();  // "Woof!"

OOP in PHP
PHP
class BankAccount {
  // Typed, visibility-controlled properties
  private float $balance = 0.0;

  // Constructor promotion (PHP 8)
  public function __construct(
    private readonly string $owner,
    float $initial = 0
  ) {
    $this->balance = $initial;
  }

  public function deposit(float $amount): void {
    if ($amount <= 0) throw new InvalidArgumentException();
    $this->balance += $amount;
  }

  public function getBalance(): float {
    return $this->balance;
  }
}
